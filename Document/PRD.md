# 모아(moa) 프로젝트 기획안 (PRD)

## 📌 프로젝트 개요

### 프로젝트명
**모아 (moa)**

### 네이밍 의미
- "모이다"의 순우리말 어근
- 사람을 모으다 / 관심사를 모으다 / 순간을 모으다
- 영문: moa (발음 용이, 글로벌 확장 가능)

### 브랜드 슬로건
1. "모아, 모이자" (행동 촉구형)
2. "우리의 순간을 모아" (감성형)
3. "관심사로 모이는 사람들" (가치 제안형)

### 핵심 가치 제안
> **"적게 보여주고, 정확하게 매칭한다"**

---

## 🎯 타겟 사용자

### Primary 타겟
**25-35세 직장인/대학생**
- 새로운 취미/관심사 탐색 욕구
- 오프라인 네트워킹 필요
- 디지털 리터러시 높음

### Secondary 타겟
**35-45세 전문직 종사자**
- 경력 개발/네트워킹
- 워라밸 중시, 의미있는 모임 선호

### 비즈니스 사용자 (NEW)
**클래스/모임 운영자 (업장 주인)**
- 쿠킹클래스, 공방, 스튜디오 운영자
- 원데이 클래스 홍보 필요
- 정기 고객 확보 목적

## 📊 시장 분석

### 경쟁사 분석

#### 1위: 소모임 (500만 다운로드)
- 매주 14,000개 정모
- 18개 카테고리 분류
- 지역 기반 검색
- **약점**: 노후한 UX, 신뢰도 관리 어려움

#### 2위: 문토 (80만 회원)
- 소셜링/클럽/챌린지 3가지 유형
- 매너 점수 시스템
- **약점**: 사용자 베이스 부족

#### 3위: Meetup.com (글로벌 1위)
- AI 기반 개인화 추천
- **약점**: 높은 수수료, 한국 시장 특화 부족

### 트렌드 분석
1. **AI 기반 개인화** - 필수 기능
2. **신뢰도 시스템** - 생존 열쇠
3. **모임 유형 다각화** - 원데이/정기/챌린지
4. **호스트 민주화** - 누구나 주최 가능
5. **오프라인 회귀** - Gen Z/밀레니얼의 오프라인 갈망

## 💡 핵심 차별화 전략

### 1. AI 큐레이션
- 하루 3개의 추천 모임만 제공 (선택 피로 최소화)
- 관심사 + 성향 + 위치 + 시간대 종합 분석

### 2. 신뢰도 우선
- 매너온도 37.5℃ 이상만 모임 개설 가능
- 노쇼 방지 예치금 시스템 (3,000원)
- 참석률 공개

### 3. 소규모 집중
- 4-8명 소규모 모임 권장 (깊은 교류)
- 대규모 이벤트보다 진정성 있는 만남

### 4. 비즈니스 모임 통합 (NEW)
- 일반 모임 + 유료 클래스 통합 플랫폼
- 업장 인증 시스템
- 클래스 홍보 및 예약 관리

### 5. 3단계 모임 여정
```
EXPLORE (탐색) → CONNECT (연결) → GROW (성장)
```
- Explore: 원데이 모임 (부담 없는 시작)
- Connect: 정기 모임 (지속적 관계)
- Grow: 프로젝트형 (목표 달성, 포트폴리오)

## 🎨 주요 기능

### Phase 1: MVP (3개월)

#### 필수 기능 (Must Have)

**1. 회원 시스템**
- 이메일/소셜 로그인 (카카오, 네이버, 구글)
- 사용자 유형 선택 (일반 사용자 / 클래스 운영자)
- 프로필 작성 (사진, 관심사 3개, 자기소개 100자)
- 본인인증 (선택, 비즈니스 사용자는 필수)

**2. 모임 시스템**
- 모임 생성 (5단계 간편 폼)
  - Step 1: 모임 유형 선택 (일반 모임 / 유료 클래스)
  - Step 2: 카테고리 선택 (10개 대분류)
  - Step 3: 제목 + 간단 소개
  - Step 4: 일시 + 장소
  - Step 5: 인원 설정 (4-8명 권장, 클래스는 제한 없음)
  - Step 6: 참가비 설정 (무료/예치금/유료)
- 모임 상세 페이지
- 참여 신청/취소

**3. 클래스 운영자 기능 (NEW)**
- 업장 정보 등록 (상호명, 사업자등록번호, 주소, 연락처)
- 클래스 일정 관리 (다회차 등록 가능)
- 예약 관리 대시보드
- 정산 내역 조회

**4. 발견(Discovery) 시스템**
- 홈 피드 (AI 추천 3개 + 새로운 모임 피드)
- 카테고리별 탐색
- 지역 필터 (내 주변 5km, 10km, 같은 구)
- 날짜 필터 (이번 주, 이번 주말, 다음 주)
- 모임 유형 필터 (일반 모임 / 유료 클래스)

**5. 신뢰도 시스템**
- 매너온도 (시작 36.5℃)
- 참석률 표시
- 간단 리뷰 (참석 후 호스트/참여자 상호 평가)
- 업장 인증 배지

**6. 커뮤니케이션**
- 모임별 오픈채팅 (참여 확정자만)
- 참석 확인 체크인 (QR 코드 or 위치 기반)

**7. 마이페이지**
- 참여 예정 모임
- 주최하는 모임
- 관심 모임
- 내 매너온도/참석률

**8. 관리자 시스템 (RBAC)**
- 슈퍼 관리자: 전체 시스템 관리
- 클래스 관리자: 본인 업장 관리
- 일반 사용자: 제한적 권한
